# Liquid UI

Welcome to **Liquid UI** - the first complete Apple Liquid Glass library for React.

Beautiful, performant, and accessible components that bring Apple's revolutionary design language to the web.

## What is Liquid UI?

Liquid UI is a collection of React components that implement Apple's Liquid Glass design system with pixel-perfect accuracy. Each component features:

- **Apple-Perfect Glass Effects** - Authentic liquid glass with real physics
- **60fps Performance** - Buttery smooth animations optimized for all devices  
- **Developer Experience** - TypeScript-first with excellent IntelliSense
- **♿ Accessibility** - WCAG AA compliant out of the box
- **Customizable** - Extensive theming and configuration options

## Core Philosophy

1. **Physics-Based** - Real Apple formulas: `1.0 - normalizedDist²`
2. **Performance First** - <50kb bundle, 60fps animations
3. **Accessibility Always** - Screen readers, keyboard navigation, color contrast
4. **Developer Joy** - Intuitive APIs, great TypeScript support

## Getting Started

```bash
npm install @liquid-ui/react
```

```tsx
import { LiquidCard } from '@liquid-ui/react'

function App() {
  return (
    <LiquidCard variant="frosted" interactive>
      <h2>Hello Liquid UI!</h2>
      <p>Beautiful glass effects with zero effort.</p>
    </LiquidCard>
  )
}
```

### Design Tokens

All components use a consistent set of design tokens based on Apple's specifications:

```ts
const liquidTokens = {
  glass: {
    blur: {
      subtle: '5px',    // Light blur
      regular: '12px',  // Standard Apple  
      strong: '20px'    // Maximum blur
    },
    opacity: {
      light: 0.15,      // 15% Apple minimum
      regular: 0.25,    // Balanced
      strong: 0.85      // 85% Apple maximum
    }
  }
}
```
